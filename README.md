## Nota
Se debe crear una carpeta -Datos- en la raiz del proyecto, ya que el codigo no lo hace.
Para agregar un nuevo producto en lo posible agregarlo P00#, siendo # un numero del 6-9 o en su defecto P0## siendo # un numero del 1-9

## Se cumple con los requisitos funcionales propuestos.
Gestión del inventario
1. Registrar, Consultar, Actualizar y Eliminar productos
2. Cada producto debe tener: Titulo, autor, categoria, precio, cantidad en stock

Registro y consulta de ventas
1. Permitir registar ventas de productos asociando: cliente, producto vendido, cantidad, fecha y descuento.
2. Validar stock disponible y actualizarlo automaticamente

Modulo de reportes
1. Mostrar el top 3 de porductos mas vendidos
2. Generar reporte de ventas totales agrupadados por autor

Validaciones avanzadas
1. Validar entradas (números positivos, campos oblogatorios, formatos correctos)
2. NO permitir ventas con stock insuficiente
